Detailed Description

template<typename CharType = char>
class rapidschema::Regex< CharType >

This class provides basic regex support for JSONSchemas.

JSONSchema regex follows the ECMA-262 regex specification http://www.ecma-international.org/publications/standards/Ecma-262.htm. The current implementation relies on the regex implementation in the stl which is not an exact implemenation of the standard, but a modified version. See https://en.cppreference.com/w/cpp/regex/ecmascript for reference.
